The goal of this section is to introduce two important tools
in stochastic control theory, i.e., the dynamic programming principle (DPP) 
and the stochastic maximum principle (SMP), by using a simple 
linear-quadratic-Gaussian (LQG) control problem as an example.

Consider the controlled stochastic differential equation (SDE):

$$
dX_t = \alpha_t dt + \sigma dW_t
$$

The goal is to find the value function $V(x,t)$ defined as:

$$
V(x,t) = \inf_{\alpha} \mathbb{E}^{x,t} \left[ \int_{t}^{T} l(\alpha_s) ds + g(X_T) \right]
$$

where the running cost $l(a)$ and terminal cost $g(x)$ are given by:

$$
l(a) = \frac{1}{2} a^2, \quad g(x) = \frac{1}{2} x^2
$$

The first approach is to use 
the Dynamic Programming Principle (DPP), 
which leads to the Hamilton-Jacobi-Bellman (HJB) equation. 
Define the Hamiltonian by Legendre transform of the running cost:

$$
H(p) = \sup_{a} (pa - l(a)) = \frac{1}{2} p^2
$$

An important property of the Hamiltonian is

$$a^* = \arg\sup_{a} (pa - l(a)) = (l')^{-1}(p) = H'(p).$$

The HJB equation is:

$$
\begin{cases}
V_t + \frac{\sigma^2}{2} V_{xx} = H(-V_x) \\
V(x,T) = g(x)
\end{cases}
$$

Taking $w = V_x$, using the fact that $H(p) = \frac{1}{2} p^2$, we 
derive the Burgers' equation:

$$
w_t + \frac{\sigma^2}{2} w_{xx} = w w_x, \text{ on } \mathbb{R} \times [0, T],
\quad w(x,T) = x \text{ on } \mathbb{R},
$$

or equivalently, with $\tau = T-t$ and $u(x, \tau) = w(x, t)$, we have

$$
u_\tau + u u_x = \frac{\sigma^2}{2} u_{xx}, \text{ on } \mathbb{R} \times [0, T],
\quad u(x,0) = x \text{ on } \mathbb{R}.
$$

Using the Riccati ansatz $w(x,t) = \phi(t)x$, we solve for $\phi(t)$:

$$
\phi'(t) = \phi^2(t), \quad \phi(T) = 1 \implies \phi(t) = \frac{1}{1 + (T-t)}
$$

The final value function is:

$$
V(x,t) = \frac{x^2}{2(1 + T - t)} + \frac{\sigma^2}{2} \ln(1 + T - t)
$$


Accordingly, the solution of the control problem can be obtained by

$$a^*(p) = p, \text{ with } p = -V_x(x, t) = -w(x,t).$$

Therefore, the optimal control is given by 

$$\alpha^*_t = a^*(-V_x(X_t, t)) = -w(X_t, t) = - \frac{X_t}{1 + T - t}.$$


The second approach is to use the Stochastic Maximum Principle (SMP), which leads to a system of forward-backward stochastic differential equations (FBSDEs).
If $(X_t, \alpha_t)$ is an optimal pair, then there exists
$(Y_t, Z_t)$ satisfying 

$$
\begin{cases}
dX_s = \alpha_s ds + \sigma dW_s, & X_t = x \\
dY_s = Z_s dW_s, & Y_T = g'(X_T)
\end{cases}
$$

with the optimality condition 

$$Y_s + \alpha_s = 0, \forall s\in [t, T].$$

By setting $Y_t = \phi_t X_t$, we recover the Riccati equation:

$$
\dot{\phi}_t = \phi_t^2
$$

The rest is the same as the DPP approach.
